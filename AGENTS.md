# AGENTS.md

## 🎯 Project Overview

AmarEla is a social impact web platform designed to promote and give visibility to services provided by women, especially in underrepresented communities.

The project is currently a **static website (HTML, CSS, JavaScript)** with simple structure and no heavy frameworks.

The goal is to evolve the project while keeping it:
- lightweight
- accessible
- easy to maintain
- community-driven

---

## 🧠 Agent Purpose

You are an AI agent helping improve the AmarEla project with focus on:

- 🎨 Modern UI/UX improvements
- ♿ Accessibility (a11y)
- 📱 Responsiveness (mobile-first)
- 🧹 Clean and maintainable code
- ⚡ Performance optimization

---

## 🚫 Constraints

- Do NOT introduce heavy frameworks (React, Angular, etc.)
- Keep the project simple and static-first
- Avoid overengineering
- Preserve the project's social purpose and simplicity

---

## 🧱 Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript
- GitHub Pages (deployment)

---

## 🎨 Design Guidelines

- Use modern, clean and minimal design
- Prefer soft color palettes (aligned with AmarEla identity)
- Improve typography hierarchy
- Add whitespace and breathing room
- Use consistent spacing system

---

## ♿ Accessibility Guidelines

- Ensure good color contrast
- Add alt text to images
- Use semantic HTML
- Improve keyboard navigation

---

## 📱 Responsiveness

- Mobile-first approach
- Ensure cards and layout adapt to small screens
- Avoid horizontal scroll

---

## 🧹 Code Quality

- Keep CSS organized (consider utility patterns)
- Avoid duplicated code
- Use clear naming conventions
- Keep JS simple and readable

---

## 🚀 Improvement Priorities

1. UI modernization
2. UX improvements (navigation, readability)
3. Accessibility fixes
4. Performance improvements
5. Code cleanup

---

## 💡 Mindset

There is no "perfect solution".

Always:
- Consider context
- Propose simple improvements first
- Iterate instead of overbuilding
